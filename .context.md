# Verdict — Contexto del proyecto

## Stack
- React 19, TypeScript 6, Vite 8, Tailwind CSS 4
- Linter: oxlint
- Tests: Vitest
- Animaciones: Framer Motion
- Fuentes: Satoshi (vía CDN Fontshare) + Space Grotesk (vía npm @fontsource-variable)

## Arquitectura
- State machine con `useReducer` (gameReducer.ts + types.ts)
- 8 fases: `waiting_start → topic_reveal → dice_roll → team_playing → team_done → second_team_playing → second_team_done → game_over`
- El juez decide en `second_team_done` con acción `JUDGE_DECIDES`
- Primer equipo en llegar a 5 puntos gana (`WINNING_SCORE = 5`)

## Patrón visual
- **Tema**: claro minimalista con fondo `#f5f2ed` y gradiente suave
- **Tarjetas**: fondo blanco (`#ffffff`), borde sutil, `rounded-xl`, `shadow-lg`
- **Color primario**: terracota `#d95c3a`
- **Color texto**: carbón `#1c1917`
- **Acentos**: esmeralda (equipo A) y azul (equipo B)
- **Botones 3D**: `shadow-[0_5px_0_var(--color-primary-shadow)]` + `active:translate-y-[4px]` + `duration-75`
- **Transiciones entre pantallas**: Framer Motion AnimatePresence, fade + slide 0.25s
- **Contador animado**: heartbeat keyframe por segundo, glow circula, cambia a danger + pulse < 10s
- **Animación del dado**: ciclo de emojis 1.2s + pausa 300ms

## Persistencia
- `localStorage` con clave `verdict-game`
- Prompt al recargar si hay partida guardada: "Continuar / Nueva partida"
- Se guarda en cada cambio de `gameState`
- Se borra al hacer "Nueva partida"
